# ukele-app

React Native mobile app built with [Expo](https://expo.dev) and TypeScript.

## Prerequisites

- Node.js 20+
- npm
- [Expo Go](https://expo.dev/go) on a device, or Xcode / Android Studio for simulators

## Getting started

```bash
npm install
npm start
```

Then press `i` for iOS simulator, `a` for Android emulator, or scan the QR code with Expo Go.

## Scripts

| Command | Description |
| --- | --- |
| `npm start` | Start the Expo dev server |
| `npm run ios` | Open on iOS |
| `npm run android` | Open on Android |
| `npm run web` | Open in the browser |

## AI tooling

This project uses [Gentle-AI](https://github.com/Gentleman-Programming/gentle-ai) with workspace-scoped Cursor config in `.cursor/`.

When you start building features, run `/sdd-init` in Cursor to register project context.
